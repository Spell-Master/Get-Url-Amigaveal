# Get-UrlAmigavel

Exemplo de como manipular URL Amigáveis quando solicitado o link via GET

## Dependência: Apache Serve

#### Como configurar
```
 - 1= Abrir: function/define.php
 - 2= Editar a linha: 'siteUrl' => 'http://127.0.0.1/Get-UrlAmigavel/', // Endereço do website
 Colocar o endereço url de onde vai está hospedo, seja local, seja remoto. (Não esquecer de colocar uma barra no final)

```
